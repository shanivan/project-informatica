Кратко о системах счисления

Основная теория

Что такое системы счисления?

Система счисления — это способ представления чисел с помощью определенного набора символов и правил.
В каждой системе есть основание (или база), которое показывает, сколько символов используется.

Основные системы счисления

Десятичная 10 (0-9)
Двоичная 2 (0, 1)
Восьмеричная 8 (0-7)
Шестнадцатеричная (16 0-9, A-F)

Как устроены числа в разных системах?

Число состоит из цифр, каждая из которых занимает определенную позицию.
Значение числа — сумма произведений цифр на основание, возведенное в степень, равную позиции цифры (считая справа налево).

Формула

Перевод чисел из десятичной системы в другую

Алгоритм

Чтобы перевести десятичное число в систему с основанием b:

  1. Делим число на b, запоминаем остаток — он будет цифрой в новом числе.
  2. Делим частное снова на b, снова запоминаем остаток.
  3. Повторяем, пока частное не станет 0.
  4. Число в новой системе — это остатки, прочитанные в обратном порядке.

Пример: перевод 156 в двоичную

156 / 2 = 78, остаток 0
78 / 2 = 39, остаток 0
39 / 2 = 19, остаток 1
19 / 2 = 9, остаток 1
9 / 2 = 4, остаток 1
4 / 2 = 2, остаток 0
2 / 2 = 1, остаток 0
1 / 2 = 0, остаток 1

Читаем остатки в обратном порядке: 10011100 — это двоичное представление числа 156.

Особенности шестнадцатеричной системы

Используются цифры 0-9 и буквы A-F (где A=10, B=11, ..., F=15).
Очень удобно для представления байтов (например, 0xFF = 255).
